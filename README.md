## 使用

- 在QQ内测群内/[App Center](https://install.appcenter.ms/users/xc-chan/apps/wearbili/distribution_groups/insider)获取安装包
- 利用您手头上的工具将安装包传输并安装到手表
- **开发者没有帮您解决安装问题/帮您安装的义务**
- **安装过程中遇到问题请善用互联网**

## 配置相关

推荐分辨率：`340x360`

推荐屏幕尺寸：`2.12‘`

推荐处理器：能跑就行

推荐内存：`2gb+`

推荐储存：`512mb+`
